# A Standalone Post

This is a standalone post. All the assets it embedded is not located locally on the server.

This blog system allows you to commit your post with standalone markdown files.

```
- content
  |
  +- _posts
     |
     +- `YEAR`-`MONTH`-`DAY`-standalone-post-1.md
     |
     +- `YEAR`-`MONTH`-`DAY`T`HOUR`_`MINUTE`Z-standalone-post-2.md
     |
     +- `YEAR`-`MONTH`-`DAY`T`HOUR`_`MINUTE`Z`+/-``HOUR``MINUTE`-standalone-post-3.md
```
